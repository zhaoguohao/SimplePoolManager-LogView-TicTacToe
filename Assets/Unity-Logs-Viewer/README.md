# Unity-Logs-Viewer
只需一步，就能在运行时查看Console的Log！


准备步骤：
1、通过菜单Reporter->Create在第一个场景创建所需物体
2、将Reporter.cs的执行顺序优先级调到最高” Scrip execution order ” in (Edit -> Project Settings )

使用方法：
使用鼠标左键按下或者手指在屏幕上画一个圈，会弹出log界面
